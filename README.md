# 🎮 Python Quest – Retro Arcade Python Trainer

Python Quest is a retro-style, desktop-based Python learning game built entirely with pure Python (Tkinter).
It combines structured lessons, interactive coding challenges, and a progression system to teach Python fundamentals in a fun, game-like experience.

## 🚀 Features
- Arcade-style progression system (XP, ranks, levels)
- Interactive coding challenges with real-time execution
- Auto-save system (progress stored locally)
- 11 structured levels covering core Python concepts
- Quizzes and challenges at the end of each level
- Runs fully offline (no internet required)

## 📦 Requirements
- Python 3.x
- Tkinter (included with most Python installations)

## ▶️ How to Run
python3 python_quest.py

## 💾 Save System
Progress is automatically saved to:
python_quest_save.json
